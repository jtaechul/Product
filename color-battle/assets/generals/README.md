# assets/generals — 장수 초상

VS 카드·순위표에 쓸 장수 초상(누끼 PNG)을 여기에 둔다

## 파일명 규칙 (고정 — 코드가 이 이름을 참조하게 됨)
| 세력 | 파일명 |
|---|---|
| 화염단 | `flame.png` |
| 심해군 | `wave.png` |
| 뇌전대 | `bolt.png` |
| 정글파 | `leaf.png` |
| 보라단 | `star.png` |
| 벚꽃단 | `blossom.png` |

## 규격
- **투명 배경 PNG** (흰 배경 누끼 제거, 머리카락 사이 흰 배경까지 제거)
- 3:4 세로, 가슴 위(chest-up) 구도
- 권장 높이 1000~1200px (VS 카드용 충분)

## 현재 상태 (완료)
- 6장 누끼 완료 배치 (896×1200 RGBA, 투명 배경)
- **누끼 방식**: AI 매팅(rembg u2net + alpha matting)으로 외곽·머리카락을 부드럽게 분리 →
  순수 흰색 연결성분을 **머리카락 틈 영역(외곽 배경 근접)에서만** 제거 → 갑옷 내부의 밝은
  하이라이트/스크래치는 보존(내부에 구멍이 뚫리지 않게).
  - ⚠️ 단순 flood-fill/밝기임계 방식은 이 초상들의 **탈색(desaturated) 강철 갑옷**을 배경으로
    오인해 갑옷을 갉아먹으므로 부적합 → AI 매팅 채택.
- VS 카드 렌더러(index.html)가 이 파일들을 자동 로드(`assets/generals/<sig>.png`). 파일이 없으면
  세력 엠블럼으로 자동 대체.
