# 천하쟁패 (Color Battle) — 설계 원칙 문서 v0.9

> 시청 심리(검증) → 설계 원칙 → 우선순위 이후 모든 개발 판단의 기준 문서
> 목표 지표: 시청 지속율 · 시청 유입율 · 추가 콘텐츠 기대율

## 0. 변경 이력
- v0.1~0.7: 시뮬레이션 프로토타입 반복 (색확산 → 지역단위 지형전)
- **v0.9: 실제 데이터로 시청 심리를 검증하고, 이를 설계 원칙·우선순위로 고정**

## 1. 근본 통찰 (데이터 검증 결과)
- 이 장르의 본질은 **"전략 게임"이 아니라 "관찰 대상"**이다
- 핵심 레버는 **응원 · 보이는 인과(투명성) · 앰비언트 · 단순반복 · 캐릭터 연속성**
- **깊은 전략 시스템은 흥행 요인이라는 근거가 없다** → 우선순위 하위
- 근거강도 표기: ◎ 강하게 확인 · ○ 부분 확인 · △ 추정(직접 근거 약함)

---

## 2. 설계 원칙 (10개)

### P1. 세력 정체성 & 응원 유도 ◎
- 근거 — 팬덤은 "무엇의 팬인지보다 집단 소속감"이 핵심, 팀 라이벌·응원 구도가 관전 스포츠를 만듦
- 실행 — 세력마다 뚜렷한 색·이름·문장·성격, **경기 시작 시 세력 소개 인트로 + "당신의 색을 골라라" 유도 자막**
- 검증 — 시청자가 특정 세력을 자기편으로 찍고 결과를 기다리게 되는가

### P2. 보이는 인과 (투명성) ◎ [최우선]
- 근거 — 매력은 "추상적 무작위를 눈에 보이는 드라마로" 바꾸는 것, 과정의 투명성이 알고리즘식 무작위가 못 주는 신뢰를 줌 (네가 반복 요구한 지점과 일치)
- 실행 — 승패의 모든 원인(행군·병력 숫자·자원 점유·전투 결과)을 **화면에 노출**, 숨은 규칙 금지
- 검증 — 소리 없이 봐도 "왜 저 세력이 이기는지" 설명되는가

### P3. 3초 가독성 (앰비언트) ◎ [최우선]
- 근거 — "뇌 마사지·가벼운 최면", 하나의 대상에 집중하는 명상적 이완이 최강 동력
- 실행 — 소리·자막 없이 **3초 내 전황 파악** (색·크기·위치만으로 우열이 읽혀야), 시선 뗐다 봐도 즉시 복귀
- 검증 — 음소거·배속으로 흘깃 봐도 판세가 읽히는가

### P4. 단순함 유지 ◎
- 근거 — 대칭·패턴·반복 선호, 단순할수록 편안한 몰입
- 실행 — 규칙은 **한 문장**으로 설명 가능("자원 먹고 정복"), UI·정보 최소화
- 검증 — 처음 본 사람이 10초 안에 규칙을 이해하는가

### P5. 리듬: 고조–이완 ○
- 근거 — 도파민·세로토닌 분비, 감정 기복이 몰입을 만듦
- 실행 — 잔잔한 확장(이완) 사이에 **이벤트(일기토·수도 함락)로 고조** 박자
- 검증 — 지루한 구간 없이 긴장의 파도가 반복되는가

### P6. 역전 보장 △
- 근거 — 무작위·여정은 확인, "리드가 계속 뒤집힘"은 추정
- 실행 — **스노볼 억제 + 고무줄**로 일방적 전개 방지, 최약체 반란 장치
- 검증 — 순위가 경기 중 여러 번 뒤집히는가 (일방적이면 실패)

### P7. 완결 신호 △
- 근거 — "끝을 봐야 끝까지 본다"는 추정
- 실행 — **진행 게이지 · 생존 세력 수 · "곧 통일" 신호**로 결말 임박을 계속 상기
- 검증 — 시청자가 "누가 이기나" 확인하려 끝까지 남는가

### P8. 정복 카타르시스 △
- 근거 — 에스카피즘 확인, "안전한 폭력" 프레임은 추정
- 실행 — 정복·멸망·편입을 **파괴적·극적 연출**(왕 얼굴·정지 컷)로
- 검증 — 멸망 순간이 그 자체로 썸네일감인가

### P9. 캐릭터 연속성 & 시리즈성 ◎
- 근거 — 마블에 이름·배경·전적이 붙고, 은퇴 마블에 팬 애도·거대 위키까지 존재
- 실행 — **장수·세력에 지속 정체성**(이름·초상·전적), 경기 후 **요약 카드**(MVP 장수·최대 역전) → 다음 편 티저
- 검증 — "그 장수/세력이 또 나왔다"는 기대가 생기는가

### P10. [반원칙] 전략 깊이 과잉 경계 ◎
- 근거 — 깊은 전략(무기 트리·명령·외교)을 흥행 요인으로 지목한 근거 없음, 오히려 P3·P4를 해침
- 실행 — 전략 기능은 **"보는 재미"에 직접 기여할 때만** 추가, 기본은 관찰 명료성 우선
- 검증 — 새 기능이 P2·P3·P4를 해치지 않는가 (해치면 보류)

## 4. 백로그 재정렬 (우선순위)

**1순위 (핵심 레버·바로 착수)**
- P3 3초 가독성 — 우열 즉시성(색 진하기·크기·경계) 개선
- P2 보이는 인과 — 남은 비가시 규칙 노출
- P6 역전 보장 — 스노볼 억제 실측·튜닝

**2순위 (유입·시리즈성)**
- P1 응원 — 세력 성격 + 시작 인트로
- P9 연속성 — 장수 고정 정체성 + 경기 후 요약 카드
- P8 카타르시스 — 멸망·역전 정지 컷(썸네일 자동 후보)

**3순위 (완성도)**
- P5 리듬 · P7 완결 신호

**보류/강등 (근거 부족)**
- 무기 강화 트리 · 명령 시스템(진격/요새/기습) · 외교 → P10에 따라 "보는 재미 기여" 입증 전까지 보류

## 5. 다음 실행 후보 (이 문서 기준)

- 유입 패키지(P1+P8+P9) — 썸네일·시리즈성, 유입·기대율 직결
- 역전 밸런스(P6) — 지속율 근본, 실측 필요
