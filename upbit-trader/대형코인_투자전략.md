# 비트코인·이더리움·리플(BTC·ETH·XRP) 투자전략 정리

> **한 줄 요약**: 대형코인은 "그냥 사서 들고 있기"도 강하지만 **폭락이 너무 큽니다(-85~95%)**.
> 그래서 **"종가가 200일 평균선 위면 보유, 아래면 팔고 현금"** 이라는 단순한 규칙으로
> **폭락을 피해 손실 폭을 절반 이하로 줄이는** 전략을 택했습니다. (수익 극대화가 아니라
> "덜 잃으며 꾸준히" = 위험 대비 수익 중심)

작성: 2026-06-13 · 관련 코드: `scripts/analyze_majors.py`(분석), `scripts/majors_trade.py`(실행봇)

---

## 1. 목표를 뭘로 잡았나
- 큰 코인(BTC·ETH·XRP)은 장기적으로 우상향했지만, 중간에 **고점 대비 -85~95% 폭락**을
  여러 번 겪었습니다. 보통 사람은 이 낙폭을 못 버티고 바닥에서 팔게 됩니다.
- 그래서 목표를 **"위험 대비 수익"** 으로 잡았습니다. 즉 *얼마 버느냐*보다
  ***얼마나 덜 깨지면서* 버느냐**를 봅니다.
- 핵심 점수: **Calmar = 연수익 ÷ 최대낙폭**. (낙폭 1만큼 감수하고 수익을 얼마 냈나 → 높을수록 좋음)

## 2. 어떻게 검증했나 (믿을 수 있게)
- **두 개의 서로 다른 데이터**로 교차 확인 — 우연·과최적화 방지:
  - 일봉: Coin Metrics, 2010년대~현재 (아주 긴 기간)
  - 시간봉: Bitfinex, 2013~2019 (완전히 독립된 다른 표본)
- 각 코인을 **앞 60%(학습) / 뒤 40%(검증)** 로 갈라, **한 번도 안 본 뒤 40%에서도**
  순위가 유지되는지 확인 (실전과 같은 조건).

## 3. 비교한 전략들
단순보유(그냥 들고 있기) / **추세필터(200일선 레짐)** / 이동평균 교차 / MACD / 볼린저 / RSI / 변동성돌파 / 잠수함 스윙

## 4. 결과 (검증구간 = 뒤 40%, 한 번도 안 본 구간 기준)
숫자는 **Calmar(위험대비수익, 높을수록 좋음)** 와 **최대낙폭(작을수록 안전)** 입니다.

| 코인 / 데이터 | 추세필터 Calmar | 단순보유 Calmar | 최대낙폭(추세필터 → 단순보유) |
|---|---|---|---|
| BTC 시간봉 | **2.81** | 0.63 | -44% → -84% |
| ETH 시간봉 | **0.59** | -0.70 | -45% → -83% |
| ETH 일봉 | **0.51** | -0.06 | -38% → -72% |
| BTC 일봉 | 0.53 (MACD가 0.74로 1위) | 0.54 | -62% → -77% |
| XRP | (추세필터 약함) | — | — (XRP는 MACD가 더 나았음) |

**공통 결론**: 어떤 추세 전략이든 **최대낙폭을 -85~95% → -40~69% 로 크게 줄였습니다.**
이게 우리가 노린 "덜 깨지며 버티기"의 핵심입니다.

## 5. 그래서 고른 전략 — 추세필터(200일선 레짐)
**규칙은 딱 이거 하나:**
- 코인의 **오늘 종가가 200일 평균선보다 위** → **보유**(상승 추세에 올라타 있기)
- 코인의 **오늘 종가가 200일 평균선보다 아래** → **전량 팔고 현금** 보관(폭락 회피)

**왜 이걸 골랐나**
1. 검증구간에서 **BTC·ETH 모두 단순보유를 이김**(가장 일관되게 우수).
2. **모든 코인에서 폭락 폭을 대폭 축소** → 목표(위험 대비 수익)에 정확히 부합.
3. **매매가 드뭅니다**(추세가 바뀔 때만) → 수수료·잦은 헛매매(휩쏘)에 강하고 규칙이 단순.

## 6. 솔직한 한계 (꼭 알아두세요)
- **강세장에선 수익이 단순보유보다 낮을 수 있습니다.** 현금으로 빠져 있는 구간이
  있기 때문 — 대신 폭락을 피하는 "맞교환"입니다.
- **XRP는 이 전략이 약했습니다.** 오래 횡보하며 평균선을 들락날락(휩쏘)해서요.
  XRP만 보면 MACD가 더 나았습니다. (그래도 폭락 회피 효과는 있음)
- 백테스트는 과거 데이터이고, **미래 수익을 보장하지 않습니다.** 체결·세금·슬리피지도 단순화됨.

## 7. 실제로 돌리는 법
모의(가짜 돈, 실제 주문 없음)로 충분히 본 뒤, 소액으로만 실거래하세요.

```bash
# 분석 다시 보기 (리포트는 data/majors_report_1D.txt, _1h.txt 에 저장됨)
python -m scripts.analyze_majors --tf 1D     # 일봉(장기)
python -m scripts.analyze_majors --tf 1h     # 시간봉(독립 검증)

# 실행 봇 — 총액을 코인 수로 균등 배분 (예: 30만 → 코인당 10만)
python -m scripts.majors_trade --invest 300000            # 모의(안전)
python -m scripts.majors_trade --invest 300000 --buffer 0.02  # 휩쏘 완화(±2% 완충)
python -m scripts.majors_trade --invest 300000 --live     # 소액 실거래
```
봇은 200일선 위/아래에 따라 사고팔고, **텔레그램으로 매수·매도 이유와 코인별
200일선 대비 위치**를 쉬운 말로 알려줍니다. (시각은 한국시간)

> 어렵죠. 그래서 **"200일 평균선 위에 있을 때만 들고, 밑으로 내려가면 팔고 쉬는"** 단순한
> 규칙을 씁니다. 큰 상승은 웬만큼 따라가면서 **큰 폭락은 피하는** 게 목적이에요. 검증해 보니
> 비트코인·이더리움에서 잘 통했고, 리플은 효과가 약했어요. "더 벌기"보다 **"덜 깨지며 오래
> 살아남기"** 전략이라고 보시면 됩니다.
