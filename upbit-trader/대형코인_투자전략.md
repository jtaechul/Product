# 비트코인·이더리움·리플(BTC·ETH·XRP) 투자전략 정리

> **한 줄 요약**: 대형코인은 "그냥 사서 들고 있기"도 강하지만 **폭락이 너무 큽니다(-85~95%)**.
> 그래서 **"종가가 150일 평균선 위(+2% 밴드)면 보유, 아래(-2%)로 깨지면 팔고 현금"** 이라는
> 단순한 규칙으로 **폭락을 피하면서 회복엔 빨리 올라타** 수익과 안정성을 함께 높였습니다.
> (수익 극대화가 아니라 "덜 잃으며 꾸준히" = 위험 대비 수익 중심. 단, 강세장 일부는 양보)

작성: 2026-06-13 (개선판) · 관련 코드: `scripts/analyze_majors.py`(분석),
`scripts/optimize_majors.py`(수익개선 탐색+검증), `scripts/majors_trade.py`(실행봇)

---

## 1. 목표를 뭘로 잡았나
- 큰 코인(BTC·ETH·XRP)은 장기적으로 우상향했지만, 중간에 **고점 대비 -85~95% 폭락**을
  여러 번 겪었습니다. 보통 사람은 이 낙폭을 못 버티고 바닥에서 팔게 됩니다.
- 그래서 목표를 **"위험 대비 수익"** 으로 잡았습니다. 즉 *얼마 버느냐*보다
  ***얼마나 덜 깨지면서* 버느냐**를 봅니다.
- 핵심 점수: **Calmar = 연수익 ÷ 최대낙폭**. (낙폭 1만큼 감수하고 수익을 얼마 냈나 → 높을수록 좋음)

## 2. 어떻게 검증했나 (믿을 수 있게)
- **두 개의 서로 다른 데이터**로 교차 확인 — 우연·과최적화 방지:
  - 일봉: Coin Metrics, 2010년대~현재 (아주 긴 기간)
  - 시간봉: Bitfinex, 2013~2019 (완전히 독립된 다른 표본)
- 각 코인을 **앞 60%(학습) / 뒤 40%(검증)** 로 갈라, **한 번도 안 본 뒤 40%에서도**
  순위가 유지되는지 확인 (실전과 같은 조건).

## 3. 비교한 전략들
단순보유(그냥 들고 있기) / **추세필터(200일선 레짐)** / 이동평균 교차 / MACD / 볼린저 / RSI / 변동성돌파 / 잠수함 스윙

## 4. 결과 (검증구간 = 뒤 40%, 한 번도 안 본 구간 기준)
숫자는 **Calmar(위험대비수익, 높을수록 좋음)** 와 **최대낙폭(작을수록 안전)** 입니다.

| 코인 / 데이터 | 추세필터 Calmar | 단순보유 Calmar | 최대낙폭(추세필터 → 단순보유) |
|---|---|---|---|
| BTC 시간봉 | **2.81** | 0.63 | -44% → -84% |
| ETH 시간봉 | **0.59** | -0.70 | -45% → -83% |
| ETH 일봉 | **0.51** | -0.06 | -38% → -72% |
| BTC 일봉 | 0.53 (MACD가 0.74로 1위) | 0.54 | -62% → -77% |
| XRP | (추세필터 약함) | — | — (XRP는 MACD가 더 나았음) |

**공통 결론**: 어떤 추세 전략이든 **최대낙폭을 -85~95% → -40~69% 로 크게 줄였습니다.**
이게 우리가 노린 "덜 깨지며 버티기"의 핵심입니다.

## 5. 그래서 고른 전략 — 추세필터(150일선 + 2% 완충밴드)
**규칙(코인별로 각각 적용):**
- 종가가 **150일 평균선보다 +2% 위로** 올라서면 → **매수/보유**(상승추세 탑승)
- 보유 중 종가가 **150일 평균선보다 -2% 아래로** 깨지면 → **전량 팔고 현금**(폭락 회피)
- 그 사이(±2% 밴드 안)면 → **직전 상태 유지** ← 경계선 잔떨림에 헛매매 안 함(휩쏘 차단)

**왜 200일선이 아니라 150일선 + 밴드인가** (6·8장 재검증 결과)
1. **150일선**: 200일선보다 짧아 폭락 후 **회복에 더 빨리 올라타** 수익이 크게 늘었습니다.
2. **±2% 완충밴드**: 평균선을 살짝 들락거릴 때 사고팔기를 반복(휩쏘)하는 걸 막아줍니다.
3. 결과: 검증구간 평균수익 **213% → 449%**, Calmar **0.36 → 0.56**, 최악 코인도 **+59% 흑자**
   (아래 8장 표). 일봉·시간봉 두 독립 데이터에서 **모두 200일선보다 우수**해 채택.

## 6. 솔직한 한계 (꼭 알아두세요)
- **강세장에선 수익이 단순보유보다 낮을 수 있습니다.** 현금으로 빠져 있는 구간이
  있기 때문 — 대신 폭락을 피하는 "맞교환"입니다.
- **XRP는 이 전략이 약했습니다.** 오래 횡보하며 평균선을 들락날락(휩쏘)해서요.
  XRP만 보면 MACD가 더 나았습니다. (그래도 폭락 회피 효과는 있음)
- 백테스트는 과거 데이터이고, **미래 수익을 보장하지 않습니다.** 체결·세금·슬리피지도 단순화됨.

## 7. 실제로 돌리는 법
모의(가짜 돈, 실제 주문 없음)로 충분히 본 뒤, 소액으로만 실거래하세요.
봇의 **기본값이 이미 개선판(MA150 + 완충2%)** 이라 옵션 없이 그냥 돌리면 됩니다.

```bash
# 분석/재검증 (리포트·결과는 콘솔 + data/majors_report_*.txt)
python -m scripts.analyze_majors  --tf 1D    # 기본 비교(일봉)
python -m scripts.optimize_majors --tf 1D    # 수익개선 탐색+검증(일봉)
python -m scripts.optimize_majors --tf 1h    # 독립 데이터 교차검증(시간봉)

# 실행 봇 — 총액을 코인 수로 균등 배분 (예: 30만 → 코인당 10만)
python -m scripts.majors_trade --invest 300000            # 모의(안전, 기본 MA150/±2%)
python -m scripts.majors_trade --invest 300000 --live     # 소액 실거래
```
봇은 150일선 ±2% 밴드에 따라 사고팔고, **텔레그램으로 매수·매도 이유와 코인별
150일선 대비 위치**를 쉬운 말로 알려줍니다. (시각은 한국시간)

## 8. "수익이 날 때까지" 재검증한 기록 (과최적화 차단)
단일 200일선을 넘어, MA 길이·완충밴드·기울기필터를 **자동으로 쓸어보며**(스윕) 수익을
높이되, **'안 본 구간(뒤 40%) + 두 독립 데이터 양쪽'에서 좋아지는 설정만** 채택했습니다.
(과거 1분봉 전략이 in-sample만 보고 골랐다가 독립표본에서 무너진 교훈)

**일봉 검증구간(뒤 40%) — 코인 3종 평균:**

| 설정 | 평균수익 | 최악 코인 | Calmar | 최악 MaxDD |
|---|---|---|---|---|
| 단순보유(기준) | 263% | -17% | 0.18 | -77% |
| 기존 200일선 | 213% | +18% | 0.36 | -66% |
| **150일선 + 완충2%(채택)** | **449%** | **+59%** | **0.56** | **-64%** |
| 100일선 + 완충5% | 623% | +17% | 0.58 | -62% |

→ 짧은 MA + 완충밴드가 수익·Calmar·낙폭을 **동시에** 개선. 100일선이 평균수익은 더 높지만
   **150일선이 "최악 코인까지 흑자(+59%)"** 라 더 견고해 채택(과최적화 위험↓).

**핵심 교훈(정직한 한계):**
- **기울기(MA 우상향) 필터**는 일봉에선 수익을 깎고 시간봉에선 도움 → 데이터마다 갈려
  **기본 끔**(옵션 `--slope`).
- 일봉·시간봉 **양쪽에서 세 코인 전부 흑자**인 단일 설정은 없었습니다(특히 XRP).
  그래도 공통적으로 ① 짧은 MA·② 완충밴드가 우수, ③ 모든 추세설정이 낙폭을 크게 축소 —
  이 세 가지는 두 데이터에서 일치해 신뢰할 수 있습니다.
- 여전히 **강세장 총수익은 단순보유보다 낮을 수** 있습니다(현금 구간과의 맞교환).
