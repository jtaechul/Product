# 🐣 왕초보용: 무료 서버에 봇 올리기 (한 단계씩)

> 어려운 말 없이, 화면에서 **무엇을 누르는지**만 적었습니다.
> 한 단계 하고 "됐어" 하면 다음으로 갑니다. 막히면 그 화면을 캡처해서 물어보세요.

## 비유로 이해하기
- 봇을 **꺼지지 않는 남의 컴퓨터(서버)** 에 올려서 24시간 돌리는 겁니다.
- 구글이 **작은 컴퓨터 1대를 평생 공짜**로 줍니다. 그걸 빌립니다.
- 한 번 켜두면 내 PC를 꺼도 봇은 혼자 계속 돕니다.

---

## STEP 1. 구글 클라우드 가입
1. https://console.cloud.google.com 접속 → 구글 계정 로그인
2. 처음이면 "무료로 시작하기" → 안내 따라가기 (카드 등록하지만 **자동 과금 안 됨**)
3. 화면 위쪽에 "내 프로젝트" 같은 게 보이면 성공

## STEP 2. 공짜 컴퓨터(VM) 만들기
1. 왼쪽 메뉴(☰) → **Compute Engine** → **VM 인스턴스**
2. **만들기** 버튼 클릭
3. 아래 4가지만 바꾸고 나머지는 그대로:
   - 이름: 아무거나 (예: `trading-bot`)
   - 지역(Region): **us-west1**  ← 공짜 조건 (꼭 미국 서부)
   - 머신 유형: **e2-micro**  ← 공짜 조건
   - 부팅 디스크 → 변경 → 운영체제 **Ubuntu**, 버전 **22.04 LTS**, 크기 **30**
4. 아래 **만들기** → 1분 기다리면 목록에 컴퓨터가 생김

## STEP 3. 그 컴퓨터에 들어가기
1. 만든 VM 줄 오른쪽의 **SSH** 버튼 클릭
2. 검은 화면(터미널)이 새 창으로 뜸 → 여기에 명령어를 붙여넣습니다

## STEP 4. 봇 설치 (아래를 통째로 복사 → 검은 화면에 붙여넣기 → Enter)
```bash
sudo apt update && sudo apt install -y python3-venv git
git clone https://github.com/jtaechul/Product.git
cd Product/upbit-trader
python3 -m venv .venv && .venv/bin/pip install -r requirements.txt
```

## STEP 5. 봇을 모의(가짜 돈)로 켜보기
```bash
.venv/bin/python -m scripts.swing_trade
```
- 글자들이 주르륵 나오면 성공! 봇이 코인을 살펴보는 중입니다.
- 멈추려면 키보드 **Ctrl + C**

여기까지가 "공짜로 봇 돌려보기"입니다.
실제 돈으로 매매하거나, 24시간 자동가동(키 입력·systemd)은 **그다음 단계**에서
천천히 합니다. 지금은 STEP 5까지만 해보세요.

---

## 자주 묻는 것
- **돈 나가요?** STEP 2 조건(us-west1 + e2-micro + 30GB)을 지키면 공짜입니다.
- **봇이 진짜 거래해요?** 아니요. STEP 5는 '모의'라 실제 주문이 안 나갑니다.
- **검은 화면이 무서워요** 그냥 복사-붙여넣기-Enter만 하면 됩니다.
