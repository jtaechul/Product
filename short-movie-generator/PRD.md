# PRD — 심해 나레이션 야생다큐 쇼츠 자동 생성 (deep_sea / narrated_wildlife)

> 상위 규칙: `CLAUDE.md` · 기술 상세: `docs/spec.md`
> 방향 v3(확정): **What On Earth TV 벤치마킹 — 나레이션 야생 다큐**. (v1 다큐·v2 ROV 미스터리는 폐기.)

## 1. 목표
- 실제 심해 생물 **단일 개체의 감정 서사**를 9:16 세로 쇼츠로 자동 생성해 팔로워·저장·수익을 키운다.
- 벤치마크: What On Earth TV(팔로워 29만/게시물 84 ≈ 게시물당 3.5천 — 나레이션 단일생물 심층의 높은 효율).

## 2. 타깃·포지셔닝
- 시청자: 심해·자연 다큐·미스터리 호기심층. 릴스 도달 → 저장·팔로우 전환.
- 톤: 깊고 거친 ASMR 나레이션 + 단어 카라오케 자막 + 동적 야생다큐 영상. 경이·몰입, 사실 기반.

## 3. 콘텐츠 전략 (확정)
- **단일 생물 감정 서사**(리스트형 지양). 길이 **40초 내외**(비용 상한 → 60초 이내로 시작).
- 서사 구조 고정: **[호기심 훅] → [시그니처 실제 행동] → [검증된 팩트 2~3개] → [감정적 마무리].**
- 대본: 현재형 단문 5~6개, 문장별 낭독 톤 태그. 사실 정확성 필수(날조 금지).
- 캡션: [훅 1줄] + [감각적 팩트 2~4줄] + [참여 질문 + 👇] + [브랜드 해시태그(#종명 포함)] + 출처 표기.

## 4. 범위 (스코프)
- **IN**: 종 자동선정(중복방지)·정보·종특정 실사 소싱·라이선스 게이트·대본·Veo 동적컷·합성·TTS·단어자막·
  앰비언트 SFX·출력/QC·콘텐츠 레코드·관리자 페이지(라이브러리/상세/편집/재생성).
- **보류/중단**: 게시물(카드뉴스) 동시제작 — 코드 보존, `generate_post=False`로 일시 중단.
- **OUT(MVP 밖)**: 다국어(초기 한국어 단독), 인스타 자동발행(Phase 4), TTS 음성 클론, 배치 대량 생성.

## 5. 성공 기준
- 한 편이 [훅→행동→팩트→마무리] 구조를 갖추고, 나레이션·단어자막이 동기되며, 사실만 담고, 9:16·유음.
- 무료 구조($0/월) 유지: GitHub Actions 제작 + 텔레그램 전송 + Release 호스팅 + Cloudflare Workers 관리페이지.

## 6. 핵심 결정 로그 (확정)
- **컴퓨트/호스팅**: 무료 GitHub Actions 유지(Cloudflare Containers·R2 유료 경로는 미도입, 옵션 보류).
- **영상 길이**: 40초(≈Veo 3~5컷). 40초 상한으로 시작(하루 Veo 10회 → 2편/일).
- **TTS 엔진**: **Gemini TTS 먼저**(무료·저비용, Charon 보이스) + 단어 타임스탬프 부재 → 자막 근사정렬.
  정밀 카라오케 필요 시 **ElevenLabs**(유료·단어정렬)로 교체 검토.
- **게시물**: 당분간 중단(릴스만 전환).
- **Veo 프롬프트**: 수심 인지형 필수(심해=어둠+형태잠금+부정제약). 스파이크 검증 완료(§docs/spec 참조).

## 7. 리스크·완화
- **Veo 형태 보존 한계**: Veo Lite가 특이형태종(덤보 귀지느러미) 해부를 완벽 보존 못함 → 형태잠금 프롬프트 +
  크게 틀어지면 재생성/종 교체. (정확성 하드룰상 허용 불가한 왜곡은 미발행.)
- **기포 잔존**: `no bubbles` 부정어를 Veo Lite가 완전히 지키진 못함 → 수심 어둠 프롬프트로 최소화.
- **자막 정밀도**: Gemini TTS 단어 타임스탬프 부재 → 근사 정렬(실용). 정밀 요구 시 ElevenLabs.
- **비용**: 40초=Veo 3~5컷 → 하루 2편. 초과 생성 방지(관리자 재생성 시 확인 모달).

## 8. 마일스톤
1. (완료) 스파이크: Veo 수심인지 프롬프트 화질·정확성 + Gemini TTS 음질·타임스탬프 실측.
2. (완료) 모듈: script/tts/subtitle + Veo 야생다큐 프롬프트 + `run_narrated` 파이프라인 + 검증.
3. (진행) 문서 3종 갱신.
4. (예정) Veo 실컷 한 편 완성·육안 QA(쿼터 여유 시), 대본 길이 40초 튜닝.
5. (예정, 별개) 인스타 자동발행(Phase 4), 게시물 재개.
