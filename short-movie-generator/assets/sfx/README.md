# 효과음(SFX) 드롭인 폴더

여기에 효과음 파일을 넣으면 파이프라인이 자동으로 사용합니다(합성 폴백을 덮어씀).

## 전환 효과음 (지도·수심 표시 → 본 영상)

- 파일명: **`dive_transition.wav`** (또는 `.mp3` `.ogg` `.m4a` `.aac` `.flac`)
- 넣는 위치: 바로 이 폴더 (`short-movie-generator/assets/sfx/`)
- 역할: 오프닝 스팅어(지도 스캔 → 해역 락온 → 수심 하강)가 끝나고 **본 영상으로 넘어가는 순간**
  재생되는 "다이브 후시(whoosh)" 효과음. 길이 0.6~1.2초 권장.
- 파일이 없으면 코드가 만든 합성 후시음이 대신 쓰입니다(무음 아님).
