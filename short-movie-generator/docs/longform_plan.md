# 롱폼(8분) 기획서 — 「深海の○○ 生き物 TOP 5」 시리즈

> 심해 생물 랭킹형 롱폼. 쇼츠 자산(실사·나레이션·HUD)을 재사용해 8분 1편으로 조립.

## 1. 콘텐츠 스펙 (확정)

1. **테마 = 그때그때 큐레이션.** 종에 테마를 고정하지 않고(문어=기이한·놀라운 둘 다 가능),
   매 편 쇼츠 구성을 고려해 5종 안팎을 골라 담는다. 테마 로테이션: 기이한 / 위험한 / 놀라운 /
   미스터리한 / 무서운 등.
2. **나레이션 대본 = 새로 뽑는다.** 쇼츠(40초 분량)를 그대로 쓰지 않고, 세그먼트당 상세 대본을
   새로 생성한다(보통 속도 낭독 + 상세 팩트). ※롱폼 성패의 핵심.
3. **첫 편 종 = 4~6종 가변.** 이미 있는 자료(시드)에서 특성 고려해 고르고, 필요 시 새로 검색해
   1~2종 추가 가능.

|---|---|---|

| 타이틀+몽타주 | 0:12–0:22 (10s) | 5종 초고속 컷(비주얼만·사실은 숨김) + 「深海の○○ 生き物 5選」 + "第5位から" |
| 글로벌 지도 오버뷰 | 0:22–0:32 (10s) | **세계지도에 오늘의 5개 지점(핀) 한 번에** 표시(예고) + 톤 세팅 나레이션. ※1회만 |

### 2-3. 핵심 원칙

- **카운트다운 5→1 (충격도 오름차순).** 1위 = 가장 턱 떨어지는 종. 종마다 성격 다르게(발광·의태·초장수·기괴형태·거대증)해 다양성 확보.
- **리스트 처짐 방지:** 매 세그먼트 끝 (e) 절벽 전환으로 다음 순위를 궁금하게.
- **콜드 오픈은 1위를 미리 살짝** 보여줘 "끝까지 볼 이유"를 첫 12초에 심는다(단, 사실·정체는 숨김).
- **지도→하강 시그니처:** 글로벌 지도(5핀)는 1회 오버뷰. 이후 **매 세그먼트가 "지도 복귀→그 종 핀→실제 수심 하강"**
  으로 시작 → 반복 리듬이 시리즈 정체성이 됨. 단 **각 하강 6~8s로 짧게**(수심 숫자·지역이 매번 실제로 달라 자연 변주).
- **★하드룰(가짜 좌표 금지):** 핀은 GPS 좌표가 아니라 **"지역 하이라이트"(예: 北東太平洋)** 수준으로만. **수심만 실제 숫자**로 하강.
- **정보 분배(확정):** (a)순위카드=종명 없이 「第X位」+실루엣 → (b)훅=충격 특징만(정체 미공개) →
  (c)도입 5초 내 정체 공개+상세 → (d)압축 스탬프로 확정. (궁금증 유지 + 답답함 방지 절충)
- **상시 코너 도감 HUD(확정):** 실사 동안 **핵심 2~3개(종명·수심·크기)만** 코너에 상시 표시(우주선 HUD).
  **나레이션 언급 시 항목이 하나씩 채워지는 스캔·로깅 연출**(배너 무시 방지). (d)에서 「ENTRY LOGGED」 압축 스탬프+쾅.
  - 조건: **모바일 가독성**(스펙 5개 욱여넣기 금지), **구역 배분**(좌상단 도감·우측 수심계·하단 자막·「第X位」),
    쇼츠에서 폐기한 HUD의 잡다함을 반복하지 않게 **미니멀·정갈**하게. (롱폼=도감/다이브로그 정체성으로 의도적 차별화)

### 2-4. 종 수에 따른 유연성 (실사 길이 제약)

- 세그먼트당 실사 사용 ≈ 58s((b)6+(c)52). **원본 소스가 약 60초**이므로 **5종이 최적**(딱 맞음).
- **4종**: 세그먼트가 ~104s로 길어져 실사 60초로 부족 → 루프·그래픽 증량 필요(품질 위험).
- **6종**: 세그먼트 ~69s로 타이트(실사 여유, 대신 종당 상세도 하락).
- → **권장: 5종(최적) · 6종(가능) · 4종(비권장, 늘림 부담).**

### 2-5. 해결됨 / 남은 것

- ✅ 정보 분배(훅↔본문): 위 2-3 확정.
- ✅ 데이터 표시: 상시 코너 도감 HUD(핵심 2~3개) + 압축 엔드카드 스탬프 확정.

## 3. 비주얼(16:9) — HUD 레이아웃 (확정)

> 레퍼런스 목업: `docs/longform_layout_mockup.png`

- **좋은 소식**: 원본이 이미 16:9(1920×1080) → 쇼츠처럼 9:16 크롭 불필요. 그레이딩만 입힘(과확대 문제 없음).
- **HUD 프레임 스타일(확정): 각진 구조 + "좌우 세로선"만**(전체 외곽선 X). 4코너에 짧은 직각 틱.
  - 반투명 다크 필(6,18,28,~200) + 시안 세로 멤버(bw 3). **핑크 컷코너 없음(폐기).**
- **구역 배분(확정):**
  - 좌상단 = **도감 HUD**(종명 서체 + 학명 이탤릭 + 水深·全長, `DEEP-SEA DOSSIER`·`SCAN`). 나레이션 언급 시 스캔 채움.
  - 우상단 = **순위**(`RANK` + 「第X位」 시안→마젠타 그라데이션). **글자 자동 맞춤·중앙정렬 → 프레임 밖 삐짐 금지.**
  - 우측 = **수심계**(얇은 readout + 현재 수심 마커). 유지.
  - 하단중앙 = **자막**(좌우선 각진 HUD 플레이트). 위치 확정.
  - 좌하단 = 워드마크 `ABYSS ・ 深淵アーカイブ`(작게).
  - 중앙 = 생물(항상 클리어).
- **색/톤**: 심해 틸 그레이딩(쇼츠와 통일), 시안 기본 + 마젠타 포인트.
- 원본 번인(NOAA 로고)은 실물에서 delogo로 제거.
