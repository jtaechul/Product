# AI 밈 이미지 라이브러리 (`assets/memes/`)

상황별 위트 있는 짤을 **한 번 생성해 저장 → 여러 영상에서 재사용**하는 라이브러리.
펀치라인(punch) 순간에 상황이 맞는 이미지를 꺼내 **글자는 렌더가 얹어** 밈 카드 배경으로 쓴다.

## 왜 이 방식인가 (규칙 근거)
- **저작권 안전**: AI 자체 생성물은 스펙 §3.2 화이트리스트. 남의 밈·움짤(Content ID 리스크)을 안 쓴다.
- **글자 왜곡 회피**: 이미지에는 **글자를 굽지 않는다.** 방금 Veo 화면 글자가 깨진 것과 같은 함정.
  텍스트(자막·펀치)는 파이프라인의 깨끗한 렌더가 얹는다 → 한글이 안 깨진다.
- **최소 생성**: 라이브러리에 있으면 재사용, 없을 때만 새로 생성.

## 이미지 규격 (나노바나나 생성 시)
- **정사각(1:1, 1080×1080)** 권장 — 중앙 밈 카드로 들어감.
- **하단 1/3(대략 아래 33%)은 비우거나 단순하게** — 여기에 흰색 펀치 글자가 얹힌다(피사체·중요요소 넣지 말 것).
- **글자를 이미지에 넣지 말 것** (한글·영문 모두). 순수 그림만.
- 미래마켓 화풍 통일: 파스텔+네온 포인트, 청량·코믹, 이모지 금지, 실제 인물 사진 금지.

## 등록 방법
1. 나노바나나로 생성한 PNG를 이 폴더에 저장 (파일명 영문·소문자·언더스코어).
2. `memes.json`의 `memes[]`에 `id`·`file`·`situations`(상황 키워드)·`desc`·`text_zone` 추가.
3. `situations`가 펀치라인 텍스트/태그와 매칭되면 자동 선택된다.

## 매칭 우선순위 (`src/video/memes.py`)
`line["meme_tag"]` 명시 > `situations` 키워드가 펀치 텍스트에 포함 > 기본(첫 항목). 라이브러리가 비면 폴백(이미지 없이 텍스트 카드).
