새 상품 등록

제휴 링크만 필수입니다. 상품 페이지의 설명·리뷰를 아래 칸에 붙여넣으면 상품명·가격·특징은 제작할 때 자동으로 뽑아냅니다.

만드는 법: 사파리에서 상품 페이지를 연 뒤(리뷰까지 한 번 스크롤) 스크린샷 → 왼쪽 아래 미리보기 → 위쪽 전체 페이지 → 공유 → 파일에 저장. 그 PDF를 여기서 선택하면 상품명·가격·특징·후기를 자동으로 읽어냅니다. 스크린샷 이미지를 복사해 이 화면에 붙여넣기(⌘V)해도 첨부됩니다.

없어도 됩니다(상품 사진으로 대체). 쿠팡 상세의 판매자 시연 영상을 iPad 화면 기록으로 녹화해 올리면 제작 시 상품 구간 배경으로 자동 사용됩니다. 개당 150MB 이하.

링크를 넣으면 후보를 만들 때 그 영상을 자동으로 추출해 라인별 후보에 제품영상으로 올려줍니다(고르면 그 라인에서 영상이 재생). ⚠️ 판매자·제조사가 올린 제품 소개 영상만 넣으세요 — 타인 창작 영상은 저작권·Content ID 위험이 있고, 플랫폼이 자동 추출을 차단하면 실패할 수 있습니다 (실패 시 텔레그램 알림 + 화면 기록 업로드로 대체).

직접 입력 (선택 — 붙여넣기 없이 등록할 때)

누르면 상품이 저장되고 곧바로 대본과 자막별 이미지 후보가 자동으로 만들어집니다(약 1~3분). 완성되면 아래 자막별 이미지 선택이 자동으로 나타납니다. 이미지를 고른 뒤 제작으로 이어집니다.

자막별 이미지 선택

상품 기획하기를 누르면 자막(문장)마다 후보 이미지가 여기에 자동으로 모입니다. 마음에 들면 탭해서 고르고(여러 장 = 슬라이드쇼), 없으면 다시 찾기나 내 파일을 올리세요. 고른 결과는 제작에 그대로 쓰이고, 자동 추천도 그 선택을 학습합니다. GIF 후보·검색: Powered By GIPHY

‘이미지 후보 다시 만들기’는 대본은 그대로 두고 이미지만 새로 찾습니다. 대본(자막)까지 마음에 안 들면 ‘전체 자막·이미지 다시 만들기’로 대본부터 새로 쓴 뒤 이미지 후보를 다시 만듭니다.

저장하면 다음 제작부터 그 상품 영상에 반영됩니다. 아무것도 안 고른 라인은 자동 소싱으로 채워집니다.

미래마켓 스토어 (프로필 링크 페이지)

영상에서는 제품을 종류·통상 명칭으로만 부르고 실제 제품명은 감춥니다. 궁금해진 시청자가 프로필 링크로 들어오면 이 스토어에서 실제 상품명과 쿠팡 링크를 봅니다. 유튜브 채널 프로필 링크와 영상 고정 댓글·설명란에 아래 주소를 넣으세요.

스토어 열어보기 ↗ (miraemarket.pages.dev)

프로필·댓글에 넣을 공식 주소: https://miraemarket.pages.dev (개인 계정명 없는 브랜드 주소 · 스토어 저장 시 자동 반영). 특정 상품으로 바로 보내려면 끝에 번호 — 예: miraemarket.pages.dev/#3 → No.003 상품으로 스크롤됩니다.

GitHub 연결 (최초 1회)

이 페이지가 저장소에 상품을 기록하고 제작 버튼을 누를 수 있게 하는 출입증(토큰)입니다. 한 번 저장하면 이 기기에 계속 유지됩니다.

토큰 발급 방법 (탭만 하면 됩니다, 2분)
  1. GitHub 새 토큰 만들기 화면 열기
  2. Token name: shorts-admin / Expiration: Custom으로 최대(약 1년) 선택
  3. Repository access: Only select repositories → jtaechul/Product 선택
  4. Permissions → Repository permissions에서 딱 2개만: Contents = Read and write, Actions = Read and write
  5. 맨 아래 Generate token → 초록색 긴 문자열 복사 → 위 칸에 붙여넣고 저장

토큰은 이 기기 브라우저에만 저장되며 저장소나 서버로 올라가지 않습니다.

Giphy 키 (GIF 검색창용 · 선택)

이미지 탭의 라인마다 GIF 검색 창(직접 검색·스크롤·탭 선택)을 쓰려면 필요합니다. 한 번 저장하면 이 기기에 계속 유지됩니다.

키 받는 방법 (무료, 3분)
  1. Giphy 개발자 대시보드 열기 (가입 또는 로그인)
  2. Create an App → API 선택 → 이름은 아무거나(예: miraemarket) → 생성
  3. 화면에 나온 API Key를 복사해 위 칸에 붙여넣고 저장

GitHub 시크릿 SHORTS_GIPHY_API_KEY와 같은 키를 써도 됩니다(자동 후보 검색용과 동일). 키는 이 기기 브라우저에만 저장됩니다.
