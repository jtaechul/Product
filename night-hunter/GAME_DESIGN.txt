NIGHT HUNTER — 게임 기획서 (다른 AI 세션 온보딩용 단일 문서)
==========================================================================

이 문서는 Night Hunter 게임의 모든 핵심 시스템/데이터/컨트롤을 한 파일에
정리한 것이다. 다른 채팅 세션에 그대로 붙여넣으면 코드를 보지 않고도
게임을 이해하고 작업을 이어갈 수 있다.

저장소  : jtaechul/Product (GitHub)
경로    : night-hunter/
배포 URL: https://jtaechul.github.io/Product/night-hunter/
최신 작업 브랜치: claude/claude-md-docs-L3RVi

1. 게임 개요
==========================================================================

· 제목       : NIGHT HUNTER (나이트 헌터)
· 장르       : 3D 형사 수사 시뮬레이션 (낮 탐문 / 밤 검거)
· 플랫폼     : 웹 브라우저 (데스크톱 + 모바일, PWA 지원)
· 엔진/기술  : 순수 HTML/CSS/JS + Three.js r128 (script 태그 로딩, 빌드 없음)
· 시점       : 3인칭 추적 카메라 (좌우 회전 + 줌 + 핀치)
· 언어       : 한국어 UI (ko)

핵심 컨셉:
분당/일산 스타일의 한국형 신도시에서 여형사 소윤 또는 하윤이 되어
 - 낮 동안 시민(NPC)과 대화하며 단서를 모으고,
 - 상점에서 보급품을 구매하고,
 - 밤이 되면 3명의 납치범을 추격해 검거한다.
범인을 잡으면 납치된 아이를 구출하고 경찰서까지 동행시켜야 클리어.

2. 게임 목표 / 승패 조건
==========================================================================

승리 조건:
  · 3명의 납치범(길동, 철수, 영수) 모두 검거
  · 구출한 아이 3명을 모두 경찰서로 동행 완료
  · 처리 함수: Minigame.triggerVictory  (js/minigame.js 582)

패배 조건:
  · 체력 0 도달  (gameState.health <= 0)
  · 처리 함수: Minigame.triggerGameOver

시간 제한 (사이클):
  · 낮 7분 (420초)  →  밤 5분 (300초)
  · 낮밤 전환은 30초간 부드러운 페이드
  · 사이클이 누적되며 "1일차" → "2일차" 식으로 일자 표기

HUD 표시 (index.html 19-36):
  · 체력      ❤ × 3
  · 시간/일자 "7:00", "1일차"
  · 코인      💰 30 (시작)
  · 검거      👮 0/3

9. 검거 미니게임
==========================================================================

데이터 출처: js/minigame.js

스타일    : Dredge(드레지) 스타일의 원형 게이지 미니게임
시작 함수 : Minigame.startMinigame  (minigame.js 73-119)
입력 처리 : Minigame.attempt        (minigame.js 146)
조작      : SPACE 또는 화면 탭

작동 방식:
  1) 원형 게이지에서 마커가 회전한다.
  2) 정해진 타겟 구간(여러 개)에 마커가 진입했을 때 SPACE/탭.
  3) 모든 타겟을 연속 적중하면 검거 성공.

난이도 (범인별):
  · 길동 : 3 타겟, 회전속도 0.55
  · 철수 : 4 타겟, 회전속도 0.75
  · 영수 : 5 타겟, 회전속도 0.95

성공 시:
  · Minigame.spawnRescueChild  (minigame.js 387) — 구출 아이 생성
  · 검거 카운트 +1, 코인 +50
  · 아이를 경찰서까지 동행시켜야 함

실패 시:
  · 체력 -1 (방탄조끼 보유 시 -0.5)

15. 게임 플레이 흐름 (요약)
==========================================================================

[1] 시작 화면
    · 캐릭터 선택 모달 (소윤 또는 하윤)
    · 수배 전단지 (3명 범인 카드)
    · "수사 시작" 버튼

[2] 낮 단계 (7분)
    · 도시 자유 탐색
    · NPC 발견 → 거리 1.5 이내 진입 → 대화
    · 단서 발견 시 H 키 또는 자동 수집 (+10 코인)
    · 상점 P → 보급품 구매
    · M 키로 수사 노트 진행도 확인 (어느 범인 단서를 더 모아야 하는지)

[3] 밤 단계 (5분)
    · 거리등 점등, 손전등 활성, 창문 글로우
    · 모든 단서 수집 완료된 범인 출현
    · 추격 시 거리 8 이하로 진입하면 적이 도주 시작
    · 따라잡으면 SPACE/탭으로 검거 미니게임 트리거
    · 미니게임 성공 → 아이 구출 → 경찰서까지 동행

[4] 클리어
    · 3명 검거 + 아이 3명 경찰서 도착 → Victory

17. 작업 시 주의사항 (다른 AI 세션을 위한 가이드)
==========================================================================

· ES 모듈 import 절대 사용 금지 — 모든 모듈은 window 전역 노출 방식
  (예: window.ChibiCharacter, window.HintSystem, window.SoundManager)
· Three.js r128 호환 코드만 사용 (다른 버전 mixing 금지)
· 새 JS 파일 추가 시 index.html에 <script> 등록 + ?v=N 버전 부여 필수
· 그래픽 생성 능력 한계: 복잡한 캐릭터/스프라이트는 외부 도구
  (Midjourney, Stable Diffusion, Blender 등)에서 만들어 가져오는 게 안전
· GLB 모델은 night-hunter/assets/models/ 에 배치, GLTFLoader 사용
· 커밋 메시지는 한/영 혼용 OK
· 매 작업 후 로컬 서버(python3 -m http.server 8000) 기동 → 브라우저 검증
  → 통과 후에야 커밋/배포
